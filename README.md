# Jan

A friendly greeting application to enhance user experience.

## Features

- Interactive greeting system
- Personalized welcome messages
- Simple and easy to use

## Usage

Run the greeting application:

```bash
python3 hello.py
```

The application will prompt you for your name and provide a personalized greeting.

## Requirements

- Python 3.6 or higher

## Example

```
$ python3 hello.py
==================================================
Welcome to Jan - Your Friendly Greeting App
==================================================

What's your name? (Press Enter for default): Alice

Hello, Alice! Welcome to Jan.

Thank you for using Jan!
==================================================
```
